## eduAPI Swagger 문서

eduAPI의 Swagger 문서를 통해 실시간으로 API 요청을 보내고, 응답을 확인할 수 있습니다.

또한, 각 API 엔드포인트에 대한 상세한 요청 매개변수, 요청 예시, 응답 예시 및 응답 스키마를 제공합니다.

아래 링크를 통해 확인할 수 있으며, API에 대한 자세한 설명은 각 API에 해당하는 API 명세 챕터를 참고하여 주시기 바랍니다.
